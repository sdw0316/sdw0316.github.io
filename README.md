# Dowon Song Homepage

Personal academic homepage for Dowon Song.

## Local Preview

```sh
python3 -m http.server 4173
```

Then open http://localhost:4173/.

## GitHub Pages Deployment

For a user homepage at `https://<owner>.github.io/`, create the GitHub repository with this exact name:

```text
<owner>.github.io
```

If the repository is private, GitHub Pages requires GitHub Pro, GitHub Team, GitHub Enterprise Cloud, or GitHub Enterprise Server. The published website itself is still intended to be publicly reachable unless private Pages visibility is configured in an eligible organization/enterprise setup.

This repository includes a GitHub Actions workflow that publishes the static site from the repository root.
