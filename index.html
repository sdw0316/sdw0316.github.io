Dowon Song (송도원)

Software Analysis Laboratory · Department of Computer Science and Engineering · Korea University

My research focuses on automated software debugging by leveraging program analysis and program synthesis. I build techniques for finding, explaining, and repairing software defects, with recent work on semantics-aware overfitting patch detection for automated program repair.

CV Publications Lab Profile Google Scholar

Research

Reliable software through semantics-aware automation

I study automated software debugging techniques grounded in programming languages, program analysis, and program synthesis.

  • Automated program repair Detecting overfitting patches and improving the reliability of generated fixes.
  • Feedback for programming assignments Generating useful, context-aware feedback for student programs.
  • Semantic bug detection Finding logical errors in functional programs with automatically generated counterexamples.
